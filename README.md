# FxWealth

FxWealth is a **real cryptocurrency investment platform** that allows users to invest, trade, and manage their crypto assets securely. Built with modern web technologies, FxWealth offers a seamless and professional experience for crypto investors worldwide.

## Features

- **Secure Investments**: Safely invest in multiple cryptocurrencies.
- **Wallet Integration**: Connect and manage your crypto wallets.
- **Real-Time Prices**: Live cryptocurrency price updates.
- **User Dashboard**: Track investments, profits, and portfolio performance.
- **Admin Panel**: Manage users, investments, and platform operations.
- **Responsive Design**: Works flawlessly on desktop and mobile devices.
- **Notifications & Alerts**: Get real-time updates on market movements and account activity.

## Technologies Used

- **Frontend:** HTML5, CSS3, JavaScript (ES6)
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **Hosting:** Vercel / Render / Custom VPS
- **Security:** JWT Authentication, Password Hashing, HTTPS Support
